five：HTML基础结构和属性？

  • HTML ： 超文本 标记 语言
  • 超文本：文本内容和非文本内容（图片、视频、音频等）
  • 标记： <单词>格式来叫做标记性语言
  • 语言：一门编程语言来做网站开发，（语法格式书写的方式）
    <header> <footer> 标记也叫做标签

写法分成两种：

  • 单标签
  • 双标签

    创建标签的快捷键tab加上对应的单词，就会快速生成单词
    标签可以上下排列，也可以嵌套
    HTML常见标签：[常见标签]http://www.html5star.com/manual/html5label-meaning/
    标签的属性：来修饰标签的，设置标签当前的一些功能
    <标签 属性='值' 属性2='值'>

six：HTML初始代码

<!DOCTYPE html>  //文档声明
<html lang="en">  // html文件最外层标签：包裹着html所有的代码   lang="en"表示是一个英文网站，lang="zh-CN"表示是一个中文网站
<head>  //头部
        <meta charset="UTF-8">  //元信息：编写网页中的一些辅助信息
         <title>网页标题</title>
</head> 
<body>  //身体
                    
</body>
</html>

seven：HTML注释

写法：// /**/ <!-- -->
意义:
1、把暂时不用的代码注释起来，方便以后使用
2、对开发人员进行提示作用
快速添加注释与删除注释：
1、ctrl + /
2、shift + alt + a

eight：标题与段落？

    标题 -> title：<h1></h1>  ...  <h6></h6>
    在一个网页中h1标题最重要，并且一个.html文件中只能出现一次h1标签
    h5、h6标签在网页中不经常使用
    段落 -> 双标签：<p></p>

nine：文本修饰标签

    强调 -> 双标签
    <strong></strong><em></em>
    区别:  1.写法和展示效果是有区别的，一个加粗，一个斜体
           2.strong的强调性更强, em的强调性更弱
         
         <sub></sub>   <sup></sup>下标文本和上标文本

    删除文本    <del></del>
    插入文本    <ins></ins>
        注：一般情况下，删除文本一般要和插入文本配合使用

11、图片标签？

img -> 单标签
src: 引入图片的地址
alt: 当图片出现问题的时候，会提示一些友好的文字
title: 提示信息
width、height: 控制图片的大小
防止网速慢加载不到图片，会导致高度塌陷。建议添加图片增加上高度宽度

16、列表标签？

    1.无序列表 -> ul li 符合嵌套的规范
        ul身上有一个type属性：改变前面标记的样式(一般都是用CSS去控制)
        查文档：http://www.w3school.com.cn/tags/att_ul_type.asp
        属性值值	描述
        disc	默认值。实心圆。
        circle	空心圆。
        square	实心方块。
    2.有序列表 -> ol li 来实现
        查文档：http://www.w3school.com.cn/tags/att_ol_type.asp
        1	默认值。数字有序列表。（1、2、3、4）
        a	按字母顺序排列的有序列表，小写。（a、b、c、d）
        A	按字母顺序排列的有序列表，大写。（A、B、C、D）
        i	罗马字母，小写。（i, ii, iii, iv）
        I	罗马字母，大写。（I, II, III, IV）

    3.定义列表 -> dl dt dd 列表项需要添加标题和对标题进行描述的内容
        <dl>：定义列表
        <dt>：定义专业属于或名词
        <dd>：对名词进行解释和描述
        练习：创建定义列表

    注:列表之间可以互相嵌套形成多层级列表

17、表格标签

    <table>：   表格的最外层容器
    <tr>：      定义表格行
    <th>：      定义表头
    <td>：      定义表格单元
    <caption>： 定义表格标题
        注: 之前是有嵌套关系的,要符合嵌套规范。
        语义化标签：<tHead> <tBody> <tFood>
        注：在一个table中tBody是可以出现多次的,但是tHead和tFood只能出现一次

19、表单标签

    <from>： 表单的最外层容器
    <input>: 标签用于搜集用户信息,根据不同的type属性值,展示不同的控件,
    如输入框、密码框、复选框等。

    input标签有一个type属性,决定是什么控件.
            type属性        含义
            text            普通的文本输入框
            password        密码输入框   placeholder提示(请输入密码)
            checkbox        复选框       disabled禁止    checked默认选中
            radio           单选框
            file            上传文件
            submit          提交按钮
            reset           重置按钮

    <textarea>：多行文本框
    <select>、<option>：下拉菜单
            selected当年被选中
            disabled丧失能力禁用
            multiple多选属性
    <label>：辅助表单
             <input type="radio" name="gender" id="man"><label for="man">男</label>
            <input type="radio" name="gender" id="woman"><label for="woman">女</label>

22、CSS的样式引入方式？

1.内联样式
    在html标签上添加<style></style>属性来实现的
2.内部样式
    在<style></style>标签内添加的样式
    注：内部样式的有点,可以复用代码
    区别：内部样式的代码可以复用、符合W3C的规范标准，进行让结构和样式分开处理
3.外部样式
    引入一个外部的css文件
    <link rel="stylesheet" href="./common.css">
    <style>
        @import url('./common.css');    /*不建议使用*/
    </style>

23、CSS中颜色表示法？

2.十六进制表示法：#000000  #ffffff
    0 1 2 3 4 5 7 8 9

3.rgb三原色表示法: rgb(255,255,255)
    取值范围0~255

24、CSS背景样式

background-color:       背景色
background-image:       背景图
background-postion:     背景定位    x(left center right)   y(top center bottom)  50%  50%
background-repect:      背景平铺方式  repect \ no-repect \ repect-x \ repect-y 

background-attachment:  背景图片随滚动条的移动方式  
                        scroll默认值(背景位置是按照当前元素进行偏移的)
                        fixed(背景位置是按照浏览器进行便宜的)

26、CSS文字样式

font-family: 字体类型
    英文字体: Arial,'Times Nes Roman'
    中文字体：微软雅黑,宋体
    衬线体与非衬线体
    注意点: 
        1.多个字体类型的设置目的
        2.引号的添加的目的

27、CSS文字大小

font-size 字体大小  默认16px
        属性取值        字体大小
        xx-small        最小
        x-small         较小
        small           小
        medium          正常(默认值)
        large           大
        x-large         较大
        xx-large        最大 

font-weight: 字体粗细
        模式：正常(normal)  加粗(bold)
        写法: 单词(normal、bold) 

font-style: 字体样式
        模式：正常(normal)  斜体(italic)
        写法：单词(normal、italic)
        注： oblique也是表示斜体
        区别：1.italic 所有带有倾斜字体的可以设置
              2.oblique 没有倾斜的属性字体也可以设置倾斜操作

28、CSS段落样式？

    text-decoration: 文本修饰
    下划线
    删除线
    上划线
        text-decoration: underline; /* 下划线 */
        text-decoration: overline;  /* 上划线 */
        text-decoration: line-through;/* 删除线 */
        text-decoration: none;/* 不添加任何装饰 */
    注：添加多个文本修饰

    text-transform 文本大小写 (针对英文段落)
        text-transform: lowercase;/* 全部转化为小写 */
        text-transform: uppercase;/* 全部转化为大写 */
        text-transform: capitalize;/* 只针对首字符转化为大写 */

    text-indent: 文本缩进
            em为单位

    text-align: 文本对齐方式
            常用对齐方式：left center right justify(两端点对齐)

    line-height: 定义行高
        什么是行高? 一行文字的高度由上边距、字体大小、下边距
        默认行高不是固定值,根据当前字体的大小不断变化
       
       取值: 1.number(px)  |  scale(比例值,文字大小成比例)
    

    letter-spacing: 字体间距
    word-spacing:   词之间的间距(只针对英文字体)

    英文和数字不自动这行的问题：
        1.word-break: break-all;   (非常强烈的折行)
        2.word-wrap: break-word;   (不是那么强烈折行,会产生一片空白区域)

29、CSS复合样式：

    复合的写法,是通过空格的方式实现的
    background: red url('./img/dog.jpg') no-repeat center center; 
    border: 1px solid red;
    font: 
        特点：最少要有两个值 size family
              weight style size family √
              style weight size family √
              weight style size/line-height family √
   
    font:italic bold 20px/50px 宋体;
    注: 尽量不要混写,如果非要混写,那么一定要先写复合样式再写单一样式。

30、CSS选择器？

1、ID选择器
        #elem{
            width: 200px;height: 200px;
        }     
        <div id="elem">这是一个块</div>
        注：1.id值多次出现是不规范的。
            2.命名规范,字母_ - 数字(命名的第一位不能是数字)。
            3.命名方式,searchButton(小驼峰) SearchButton(大驼峰) 、下划线式、短线式。
                短线写法：search-small-button
              下划线写法：search_small_button
2、CLASS选择器
        .elem{
            width: 200px;height: 200px;
        }
        <div class="elem">这是一个块</div>
        
        优点：可以多次调取使用
                
        p.box{
            background-color: grey;
              }
        <p class="box">这是一个段落</p>

8、伪类选择器   
    :link   :visited    :hover  :active

    link    访问前的样式 (只能添加给a标签)
    visited 访问后的样式 (只能添加给a标签)
    hover   鼠标移入时的样式 (可以添加给所有的标签)
    active  鼠标按下时的样式 (可以添加给所有的标签)
